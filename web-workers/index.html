Web Worker Examples

Source code here: https://github.com/euclid1990/web-fundamentals

  1. Number crunching 

    The main document spawns a worker to compute prime numbers, and progressively displays the most recently found prime number.

  2. Shared workers 

    Connect to a worker and a worker can send a message back to the page when it connects to it. Received messages are displayed in a log.

  3. Image processing 

    Processes image when added to input file. Including resizing and drawing image in cavas, applying a filter (using a web worker).
